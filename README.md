Descrição:
Basicamente o jogo funciona:
Nivel 1: Nível Fácil, o computador escolhe um número aleatório entre 1 e 10. O jogador tem 3 tentativas para descobrir qual é o número. A cada erro, o jogo informa se o palpite foi muito baixo ou muito alto, ajudando o jogador a encontrar a resposta correta
Nivel 2: Nível Médio, o computador escolhe um número aleatório entre 1 e 100. O jogador tem 5 tentativas para adivinhar o número secreto. Após cada erro, o jogo informa se o palpite está abaixo ou acima do número correto, permitindo que o jogador ajuste suas próximas tentativas. A faixa maior de números torna o desafio mais difícil do que o nível Fácil
Nivel 3: Nível Impossível, o computador escolhe um número aleatório entre 1 e 1000. O jogador dispõe de 10 tentativas para adivinhar o número secreto. Após cada erro, o jogo informa se o palpite foi muito baixo ou muito alto. Como a faixa de números é muito ampla, este é o nível mais difícil do jogo e exige estratégia para encontrar o número correto dentro do limite de tentativas

Como jogar:
1- Abra o terminal
2- Digite python adivinhe.py
3- E escolha o nivel do 1 ao 3 e tente adivinhar
